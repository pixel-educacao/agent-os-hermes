# Criar conteúdo

## Quando usar

Use para transformar uma ideia, referência ou tese em conteúdo publicável.

## Passo a passo

1. Identificar a tese forte.
2. Definir público e promessa.
3. Escolher formato.
4. Criar versão 1.
5. Cortar abstração e jargão.
6. Revisar hook, ritmo e fechamento.

## Checklist de qualidade

- [ ] Existe uma tese clara?
- [ ] O primeiro bloco prende atenção?
- [ ] Há exemplos concretos?
- [ ] O texto soa humano?
- [ ] O CTA é específico?

## Limites

Não publicar automaticamente. Entregar rascunho para aprovação.
