# Salvar contexto

## Quando usar

Use para persistir informação relevante no lugar certo.

## Passo a passo

1. Classificar o tipo de informação:
   - ideia;
   - decisão;
   - tarefa;
   - aprendizado;
   - referência;
   - procedimento.
2. Consultar `PROPAGATION.md`.
3. Escrever no arquivo correto.
4. Se for procedimento recorrente, criar skill em vez de só memória.
5. Confirmar em uma frase onde foi salvo.

## Destinos comuns

- Ideias: `memory/ideas.md`
- Decisões: `memory/decisions.md`
- Tarefas: `memory/tasks.md`
- Aprendizados: `memory/lessons.md`
- Procedimentos: `skills/`

## Verificação

- [ ] A informação foi salva em arquivo certo.
- [ ] Não contém segredo.
- [ ] Não duplicou fonte de verdade.
