# Pesquisa rápida

## Quando usar

Use quando a resposta depende de informação atual ou fonte externa.

## Passo a passo

1. Transformar o pedido em 1–3 perguntas de pesquisa.
2. Buscar fontes públicas confiáveis.
3. Separar fato, inferência e recomendação.
4. Citar links importantes.
5. Concluir com ação prática.

## Formato de resposta

```md
## Resposta curta
...

## Evidências
- Fonte 1:
- Fonte 2:

## Minha leitura
...

## Próximo passo
...
```

## Verificação

- [ ] Pelo menos uma fonte viva foi consultada.
- [ ] Não há afirmação factual sem base.
- [ ] A recomendação está separada dos fatos.
