# Exemplo — Agent OS para consultor

Este exemplo mostra como adaptar o Agent OS para um consultor independente.

## Skills iniciais

- rotina diária;
- preparar reunião;
- criar proposta;
- salvar contexto de cliente.

## Rails

Pode fazer sozinho:

- organizar notas;
- criar rascunhos;
- pesquisar empresas;
- montar checklist.

Precisa perguntar antes:

- enviar proposta;
- mandar email;
- alterar contrato;
- publicar conteúdo.
